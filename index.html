Protocols

Protocols define how to implement specific functionality so that the type offering it can be used with other existing types.

Unbelievably Fast

Emojicode has incredible performance! Benchmarks¹ show that Emojicode can execute certain programs nearly 2x as fast as Java and about 10x faster than Ruby.

Low Overhead

Above that, Emojicode is tiny. The prebuilt binaries are just about 500KB. Emojicode can be built in a few minutes without any dependencies and is installed within seconds, literally.

Portable

Not to forget, Emojicode binaries are system independent and work everywhere the Real-Time Engine work – macOS, Linux, and, yes, even Windows.
